# PDF Proyectos Críticos compacto V004

## Cambio realizado

Se ajustó únicamente la exportación PDF de Proyectos Críticos para usar el mismo criterio visual aprobado en Equipos Críticos.

## Formato

- Título: `BLT Brilliant — Proyectos Críticos`.
- Encabezado compacto con fecha, período y mínimo de fallas BLT.
- Total de proyectos críticos en una segunda línea.
- Paleta aprobada de Equipos: título azul, encabezado de tabla rojo y filas alternas rojo claro.
- Nombre de archivo diferenciado: `Proyectos_Criticos_YYYY-MM-DD.pdf`.

## Columnas

1. Zona.
2. Proyecto.
3. Equipos activos.
4. Fallas BLT.
5. Equipos críticos.
6. MTBC Año Actual.
7. MTBC U365.

- `modules/equipos-criticos/equipos-criticos.js`
- `index.html`
