# Barra lateral contextual V0.1.5

## Corrección

- Se eliminó la modificación por JavaScript que reemplazaba todo el contenido de los encabezados de grupo con `textContent`.
- Se conserva la estructura interna de cada encabezado (`icono`, `título`, `flecha`).
- Al contraer la barra solo se muestran los emojis, como en el diseño original.
- Se aplicó el orden: Inicio, Operación, Portafolio, Ventas, Logística, Instalaciones, Cobranza, Usuarios y Panel de Control.
- Ventas usa el icono 🤝 y Panel de Control usa ⚙️.
- Se actualizó la versión de caché de `base.css` y `core/app.js`.
