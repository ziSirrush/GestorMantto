# Barra lateral contextual V0.1.2

## Ajuste aplicado

- En escritorio, cuando la barra lateral está contraída, el primer clic sobre cualquier parte interactiva de la barra la expande.
- El botón de expansión situado junto a Inicio se conserva sin cambios.
- Si el clic se realiza sobre un grupo contraído, al expandirse se abre ese grupo y se mantienen cerrados los demás.
- Si el clic se realiza fuera de un grupo, se restaura el grupo correspondiente a la ruta activa.
- El primer clic se reserva para expandir la barra; esto evita que el elemento navegue y vuelva a contraerla inmediatamente. El segundo clic ejecuta la navegación normal.
- En dispositivos móviles se conserva el comportamiento existente del panel deslizante.
