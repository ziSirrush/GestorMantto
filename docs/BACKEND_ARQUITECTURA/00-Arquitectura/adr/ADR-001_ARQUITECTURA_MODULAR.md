# ADR-001 Arquitectura Modular

Estado: Aprobado

## Contexto
El backend había concentrado múltiples responsabilidades en pocos controladores, dificultando mantenimiento y evolución.

## Decisión
Adoptar una arquitectura modular donde cada módulo sea dueño de su funcionalidad.

## Consecuencias
- Mayor mantenibilidad.
- Escalabilidad.
- Menor acoplamiento.
- Facilita pruebas y documentación.
